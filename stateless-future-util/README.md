## Installation

Put these lines in your `build.sbt` if you use [Sbt](http://www.scala-sbt.org/):

    libraryDependencies += "com.qifun" %% "stateless-future-util" % "0.5.1"

`stateless-future-util` should work with Scala 2.10.3, 2.10.4, or 2.11.x.
